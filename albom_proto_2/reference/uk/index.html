Програма “Коментарі до зображення” призначена для збереження метаданих в зображенні. Файл який містить метадані повинен мати формат: jpg, jpeg, bmp, gif, png, tif. Файл може містити декілька метаданих, які повинні належати різним користувачам. Метадані містять шлях де буде розміщено файл для вибраного користувача.

Програма має такі режими роботи:

  • довідка про программу;

  • режим вибору та переміщення файлів, за метаданими;

  • внесення, збереження та видалення з обраного файлу метаданих. Для переходу в даний режим необхідно виділити зображення та виконати подвійний клік мишкою, або вибрати команду контекстного меню "Змінити розташування (Metadata)".


    1. Меню режима - вибір та переміщення файлів




Команди меню (зліва направо):

- інформація про автора ;

- вибір папки, яка містить зображення ;

- масштаб показу ескіза зображення ;

- переміщення зображення згідно метаданих ;

(переміщення вибраного зображення здійснюється командою контекстного меню)

- режим “Довідка про программу”.


    1. Меню режима - внесення, збереження та видалення метаданих



Команди меню (зліва направо):

- збільшення зображення (масштаб 200%) ;

- показ орігінального зображення ;

- поворот зображення по часовій стрілці ;

- запис метаданих до зображення ;

- видалення метаданих в зображенні .
